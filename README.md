
# Bank Management System (C++)

A console-based Bank Management System developed using C++.

## Features

- Create Account
- Deposit Money
- Withdraw Money
- Balance Enquiry
- Modify Account
- Delete Account
- Search Account
- View Transaction History
- Export Transactions to CSV
- Auto-generated Account Numbers
- Persistent File Storage

## Technologies Used

- C++
- File Handling
- STL (Vector, String, Algorithm)
- OOP Concepts

## How to Run

Compile:

g++ BankManagementSystem.cpp -o BankManagementSystem

Run:

./BankManagementSystem
